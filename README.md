# «Наше место» — романтическая мини-игра для Салимы ❤️

Это полностью автономный сайт для GitHub Pages. Firebase, сервер, Java, Tomcat и регистрация в сторонних сервисах не требуются.

## Что находится в проекте

- 16 тематических категорий;
- 202 вопроса о любви, воспоминаниях, быте, будущем, доверии, путешествиях, свиданиях и близости;
- обязательные ответы только внутри выбранной категории;
- сохранение черновика и прогресса в браузере;
- режимы «Мне грустно», «Мне скучно», «Хочу стать ближе» и «Удиви меня»;
- убегающая кнопка «Нет» в вопросе «А ты меня любишь?»;
- фоновые фотографии, фотокарточки и меняющиеся тёплые фразы;
- прикрепление фотографий, видео и PDF;
- создание одного файла `.love` с сохранёнными ответами;
- отдельная админ-панель для импорта и просмотра ответов по категориям.

## Как добавить ваши настоящие фотографии

В папке:

```text
assets/photos/
```

находятся восемь файлов:

```text
photo-01.jpg
photo-02.jpg
photo-03.jpg
photo-04.jpg
photo-05.jpg
photo-06.jpg
photo-07.jpg
photo-08.jpg
```

Сейчас там стоят романтические заглушки. Замени их вашими фотографиями, сохранив **точные имена файлов**. Лучше использовать вертикальные изображения 4:5 или 9:16. Сайт сам будет менять их на фоне и показывать в боковых фотокарточках.

## Как изменить тёплые фразы

Открой файл:

```text
content.js
```

В массиве `loveQuotes` находятся все фразы, которые появляются на сайте. Можно менять их, удалять или добавлять новые сообщения из вашей переписки.

## Как изменить вопросы

Все категории и вопросы находятся в файле:

```text
questions.js
```

У каждого вопроса есть категория, текст, тип ответа и обязательность. Файловые вопросы сделаны необязательными, остальные вопросы внутри выбранной категории обязательны.

## Как опубликовать на GitHub Pages

1. Создай новый публичный репозиторий на GitHub.
2. Загрузи **содержимое** этой папки в корень репозитория.
3. Открой `Settings` → `Pages`.
4. В разделе `Build and deployment` выбери `Deploy from a branch`.
5. Выбери ветку `main` и папку `/root`.
6. Нажми `Save`.
7. Через некоторое время GitHub покажет адрес сайта.

Главная страница:

```text
https://ТВОЙ-ЛОГИН.github.io/РЕПОЗИТОРИЙ/
```

Админ-панель:

```text
https://ТВОЙ-ЛОГИН.github.io/РЕПОЗИТОРИЙ/admin.html
```

## Как получать ответы

1. Салима выбирает категорию и отвечает на все обязательные вопросы этой темы.
2. Ответы автоматически сохраняются в браузере её устройства.
3. После категории она нажимает «Отправить ответы Бекназару».
4. На телефоне откроется системное меню отправки. Она выбирает Telegram или WhatsApp.
5. Если меню отправки недоступно, файл `.love` автоматически скачивается.
6. Ты открываешь `admin.html`, вводишь пароль и нажимаешь «Импортировать ответы».
7. Ответы отображаются по категориям вместе с прикреплёнными файлами.

## Важные особенности автономной версии

- GitHub Pages не принимает и не хранит ответы посетителей.
- Ответы хранятся локально на устройстве Салимы, пока она не отправит файл.
- Импортированные ответы хранятся локально в браузере твоего устройства.
- Прикреплённые файлы нельзя сохранить после перезагрузки страницы до экспорта, поэтому их лучше выбирать непосредственно перед завершением категории.
- Очистка данных браузера удалит локальный прогресс и импортированные анкеты.
- Пароль в статическом сайте защищает от случайного входа, но не является серверной системой безопасности.
